#  Level-Up

Welcome to my **first ever web project**! I'm really excited to share this with you.

I'm Han Joseph Anoop, a BS-MS student at IISER Pune. My primary focus is on theoretical physics and astrophysics research, but I built this site as a personal project to stay organized.

### Why I built this 
I built this project to **practice website building** and create a functional **checklist for my workout routine** and skincare habits. 

It was a great way to learn how to integrate tools like **Firebase** for cloud syncing while building a system that helps me track my progress toward my fitness and health goals.

### Key Features 🎮
* **Daily Checklist:** Keep track of your protein intake, caloric surplus, and specific calisthenics workouts.
* **Experience System:** Earn EXP for every task completed and level up your consistency.
* **90-Day Calendar:** A visual grid to track your journey over a full three-month protocol.
* **Cloud Sync:** Create an account to sync your progress across multiple devices.
* **Unlockable Rewards:** Earn a "Cheat Meal" reward once you hit the 500 EXP milestone.
